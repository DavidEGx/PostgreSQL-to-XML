PostgreSQL-to-XML by DavidEGx

Converts a PostgreSQL composite into a XML text

This project provides functions to translate a PostgreSQL composite type into a XML.

Suppose you have the following composite types in your database: 
        
CREATE TYPE address AS
(
    street      text,
    postal_code integer,
    city        text,
    country     text
);
        
        
CREATE TYPE customer AS
(
    customer_id integer,
    email       text,
    name        text,
    surname     text,
    address     address
);
Then running: 
        SELECT composite_to_xml(my_customer)
Gives you this result: 
    <xml>
    <customer_id type="int4" array="0">1000</customer_id>
    <email type="text" array="0">yourcustomer@email.com</email>
    <name type="text" array="0">CustomerName</name>
    <surname type="text" array="0">CustomerSurname</surname>
    <address type="address" array="0">
        <street type="text" array="0"></street>
        <postal_code type="int4" array="0"></postal_code>
        <city type="text" array="0"></city>
        <country type="text" array="0"></country>
    </address>
</xml>

Dependencies

PostgreSQL 9.1 (Also it is possible it work on old versions)

Install

Just run the install.sql into your PostgreSQL server

		$ psql -h your_server -d your_database -U your_user < install.sql
